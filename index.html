System Documentation & Limitations

Wallet Logic

Current balance is fixed at $50,000 for simulation. Equity is calculated as Balance + (Σ Positions PNL). Margin reflects total notional entry value.

Order Book Accuracy

Currently uses randomized visual data mapped to the current price. Integrating Binance @depth streams would provide real-time liquidity visualization.

Market Data

Price and 24h Change are dynamic. Volume is currently a placeholder; upgrading to a REST API fetch would allow for 24h rolling volume updates.

Trading Constraints

This is a visual simulator. It lacks wallet validation (you can trade more than you have), leverage settings, and liquidation logic.

Chart Customization

Added Interval selector. Future iterations could include a toggle for the Drawing Toolbar and Comparison Tools via the TradingView Widget API.
